# Frontend (Next.js)

- Uses App Router.
- Set NEXT_PUBLIC_BACKEND_URL to your signaling server URL.

Run:

```bash
npm install
npm run dev
```

Deploy:
- Use Vercel. Add `NEXT_PUBLIC_BACKEND_URL` in Vercel project env vars pointing to your Railway backend URL.
